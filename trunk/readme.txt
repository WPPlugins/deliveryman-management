This plugin manages deleveryman for woocommerce Product orders.

== Description ==
This plugin allows to create deliveryman and assign them to orders. 
You can:
* Manage deliveryman.
* Filter DeliveryMan by date and Delivery status.
* Generate report.

== Installation ==
1. Install plugin by searching or uploading its zip.
1. Activate plugin
1. Go to Users->Add New, from wordpress main left menu
1. Create a deliver man (user) and select 'Deliveryman' from Role drop down
1. Edit any WooCommerce order and Select Rider from the options in right side and then save order

**Note**: You must have WooCommerce activated before using this plugin.

== Frequently Asked Questions ==

= 1. How do we add order to a delivery man.

Go to user->edit User, assign user role as deliveryman, This user will be shown as delivery man in a order details.
